Switches and Connections

The control panel top includes the following switches:

SWITCH LOCATION FUNCTION
Main Power Switch Control panel top upper left Turns the control panel on and off.
ROV Power Switch Control panel top central left Turns the ROV power on and off. Twist in the direction of the arrows to turn on the ROV power and press the switch to turn it off.
LIM Module Control panel top central right Test and Reset buttons can be found on the LIM.

The control panel top includes the following connections:

CONNECTION LOCATION FUNCTION
Power (100-240 Volts AC, 50, 60 Hz) IEC male Used to connect the control panel to a power source.
USB Ports (2) Type A female Can be used to connect USB devices to the computer via the control panel.

The control panel side includes the following connections:

CONNECTION TYPE FUNCTION
Tether Whip (Specifications) 8 pin round female Used to connect the control panel to the tether for power, communications, video and accessory support.
Monitor Power Provides 12 Volts DC. Used to provide power to the monitor.
Monitor Signal HDMI Female Used to provide the video signal to the monitor.
USB Port Type A female Can be used to connect USB devices to the computer via the control panel.
Ethernet RJ-45 female Can be used to connect the control panel to the computer for Ethernet based ROV accessories.
