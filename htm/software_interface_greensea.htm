Greensea Systems Interface Overview

Greensea's VideoRay Balefire is an optional topside control software for the MSS.

To Start VideoRay Balefire, double click on the MSS GSS Server icon and then double click on the MSS GSS Client icon.

For more information about Greensea's VideoRay Balefire software, see the Greensea Knowledge Base (Login Required).

To obtain a login for the Greensea Knowledge Base, contact support@greenseainc.com or call +1 802-434-6080.
