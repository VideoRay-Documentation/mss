Greensea Systems Interface Overview

Greensea's EOD Workspace is the topside control software for the MSS.

To Start VideoRay EOD Workspace, double click on the $$config_name icon on the desktop.

For more information about Greensea's VideoRay EOD Workspace software, see the Greensea Operator's Manual. For the most recent information, visit Greensea's Knowledge Base (Login Required - free to request).

To obtain a login for the Greensea Knowledge Base, contact support@greenseainc.com or call +1 802-434-6080.
