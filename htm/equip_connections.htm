Connections Summary

The following connections are required:

  1. Connect the female end of the tether to the ROV.
  2. Secure the tether strain relief to the ROV.
  3. Connect the male end of the tether to the User Control Console.
  4. Connect the keyboard to a USB port on the User Control Console.
  5. Connect the hand controller to a USB port on the User Control Console.
  6. Connect the power cord to a suitable power source.
