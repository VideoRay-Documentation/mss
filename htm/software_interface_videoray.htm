VideoRay Interface Overview

There are four applications that are required to operate the VideoRay vehicle control software. These are stored in the ~/Flighthack folder.

  • Start the thruster controls using vr_engage.py
  • Start the camera controls using vr_engage_camera.py
  • Start the video window
    Video is automatically in record mode when the video window is started.
    When stopping video, you must enter CTRL-C in the text window to close the file properly.
  • Start the manipulator controls (optional) using vr_engage_manip_rotating.py

Each application may be started in any sequence, but following a consistent sequence such as above is recommended.
