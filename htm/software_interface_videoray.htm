VideoRay Interface Overview

These instructions are interim. Future releases of the control software will include updated instructions.

There are four desktop icons that are required to operate the vehicle control software:

  • Start the thruster controls
    There are two thruster control program icons. One (labeled vr_engage_200)is for use with 200 m or less tether, and the second (labeled vr_engage_winch) is for use with 200 m or more tether.
  • Start the camera controls
  • Start the manipulator controls (optional)
  • Start the video window
    Video is automatically in record mode when the video window is started.
    When stopping video, you must enter CTRL-C in the text window to close the file properly.

Each application may be started in any sequence, but following a consistent sequence such as above is recommended.
