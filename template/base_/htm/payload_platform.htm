Modular Payload Platform

The Modular Payload Platform is removable from the ROV and can be used to mount accessories including manipulator, sonar, camera and USBL beacon.

The Modular Payload Platform includes ballast weight slots on the sides. To install or remove ballast, loosen (but do not remove) the thumb screws that hold the covers to the platform, and slide the covers down. Return the covers and tighten the screw when finished.

Installation and Removal

To install the Modular Payload Platform, align it under the ROV and secure it with the four wing bolts. Reverse the process to remove it.
